# 《了不起的我》心智模式分析镜

> 用陈海贤的心理学框架，看穿一个人的思维底层操作系统（五层完整版）

---

## 第一层：行为层 —— 改变与行动

### 详细解释

这一层分析的是：**TA想不想改变？是什么在阻止TA改变？**

| 类型 | 具体解释 | 在素材中的表现 |
|------|----------|--------------|
| **心理舒适区** | 心理学概念，指维持现状会带来安全感 | "我习惯了""以前就是这样""还是这样吧" |
| **改变受阻** | 想改变但迈不出步，像前面有堵墙 | "我想改但改不了""下次再说" |
| **环境场** | 在某种环境下 automatism 会做某事 | "一到自习室就想睡""在家不想学习" |
| **情感驱动** | 用什么情感来推动自己/他人 | 焦虑驱动（自责/恐惧） vs 爱驱动（期待/欣赏）|

**判定方法**：看素材中TA是否想改变？阻碍是什么？

**关键书籍原文**：
> "你的心理系统就像一个鱼缸，而你就像鱼缸里的那条鱼，你在里面游看起来很自由，其实，你是被限制住了。"（第3章）

## 第二层：思维层 —— 心智模式

### 详细解释

这一层分析的是：**TA用什么方式看待世界和处理信息？**

| 类型 | 子类型 | 具体解释 | 在素材中的表现 |
|------|------|----------|--------------|
| **成长型** | — | 认为能力可以成长，失败是学习机会 | "下次我会做得更好""我能从中学到什么" |
| **防御型** | — | 认为能力固定，回避挑战保护自我 | "我就是这样，改不了了" |
| | **僵固型** | 特别在意"聪明"形象，怕显得笨 | "万一做不好别人会怎么看我""我一定要证明自己不笨" |
| | **应该思维** | 用"应该""必须"要求世界，非黑即白 | "你应该...""我必须...""不是对就是错" |
| | **绝对化** | 把一件事扩大到所有事，给判断加"永远/总是" | "你总是迟到""我永远做不到""所有人都会..." |
| **成长工具** | 与自我辩驳 | 当有负面想法时，用证据/逻辑反问 | "我真的不行吗？有什么证据？" |
| | 控制两分法 | 分清能控制vs不能控制的 | 焦虑时问"什么是，我能控制的？" |
| | 正念思维 | 用具体问句，不用抽象 | "什么场合会紧张？"vs"我性格内向怎么办" |

**判定方法**：
- TA面对失败/挑战时第一反应是什么？
- TA语言中有没有"应该""必须""总是""永远"这类词？
- TA做事是为了目标还是为了证明自己？

**关键书籍原文**：
> "僵固型思维的特点是担心犯错。担心犯错，其实就是在维护自己不会犯错的形象。"（第12章）
> "应该思维的本质，是用社会规则、他人期待来代替自发行动。"（第14章）

## 第三层：关系层 —— 自我在关系中

### 详细解释

这一层分析的是：**TA在关系中是什么角色？TA如何处理关系？**

| 类型 | 具体解释 | 在素材中的表现 |
|------|----------|--------------|
| **关系中的角色** | 同一人在不同关系中表现不同 | 对老师/对朋友/对恋人有完全不同态度 |
| | 照顾者 | 总是照顾别人、关心别人需求 | "你先吃""我帮你" |
| | 被照顾者 | 依赖别人、等别人安排 | "你决定吧""我不知道" |
| | 控制者 | 要别人按TA的方式来 | "你必须听我的""按我说的做" |
| | 追随者 | 总是跟随别人意见 | "都行""你說了算" |
| **依恋类型** | 在亲密关系中的安全感来源 | — |
| | 安全型 | 能独处也能亲密，相信对方 | 关系中放松自在 |
| | 焦虑型 | 担心被抛弃，频繁确认 | "你爱我吗""你怎么不回我" |
| | 回避型 | 逃避亲密，保持距离 | "我很忙""先不提了" |
| **关系问题** | 责任混淆的几种方式 | — |
| | "都是你的错" | 把关系问题归因对方个性 | "他就是这样的人" |
| | "都是我的错" | 为别人情绪过度负责 | "我应该让TA开心" |
| | 关系纠缠 | 爱恨交织分不开 | "我又恨他又离不开TA" |

**判定方法**：
- TA对不同人是否不同态度？
- TA在亲密关系中是什么模式？
- TA是否在替别人承担责任？

**关键书籍原文**：
> "决定你行为的不是你的个性，而是你所处的关系。"（第23章）

## 第四层：转折层 —— 人生的转变

### 详细解释

这一层分析的是：**TA是否经历过重大转变？现在处于哪个阶段？**

| 阶段 | 具体解释 | 信号词/表现 | 案例 |
|------|----------|-------------|------|
| **结束期** | 旧的身份/关系/工作在结束 | "不干了""放弃了""结束了""离职了" | 被裁员后刚离职 |
| **迷茫期** | 旧的已结束，新的还没来，容易空虚没意义 | "不知道干嘛""没意思""就这样吧""我该怎么办" | 刚辞职还没找新工作 |
| **重生期** | 找到了新的身份和方向 | "重新开始""新生活""我想通了""找到了" | 找到新工作开始新篇章 |
| **稳定期** | 暂无明显转折 | 生活稳定，没有大变化 | — |

**转折叙事分类**：
- 外归因：把转折归因为"运气不好""被逼的""环境不好"
- 内归因：把转折归因为"自己选择的""我决定的"
- 赋能型：把转折写成成长故事，"这段经历让我学到了..."

**判定方法**：
- TA近期有重大变化吗？（换工作/分手/搬家等）
- TA如何描述自己的转折？用的是什么故事？

## 第五层：人生阶段层 —— 一辈子的课题

### 详细解释

这一层分析的是：**TA处于什么人生阶段？这个阶段的核心课题是什么？**

| 阶段 | 年龄 | 具体解释 | 核心课题 | 失败风险 |
|------|------|----------|----------|----------|
| **青春期** | 12-18 | 建立自我身份认同 | "我是谁""我想成为什么样的人" | 角色混乱，不知道自己是谁 |
| **成年早期** | 25-35 | 建立亲密关系和职业认同 | 有爱人+有事业方向 | 孤独（没亲密关系）+ 空虚（没工作目标） |
| **中年期** | 35-60 | 繁衍vs停滞 | 帮助下一代/创造有价值的东西 | 感觉"一眼看到头"、无聊、混日子 |
| **晚年** | 60+ | 整合一生 | 接纳自己的一生 | 后悔、遗憾、绝望 |

**判定方法**：
- TA年龄大约多少？
- 这个年龄的核心议题TA完成了吗？
- 有没有落后于这个阶段的焦虑？

**关键书籍原文**：
> "青春期获得了稳定的自我，成年早期建立亲密关系和职业认同，中年期发展繁衍，年晚年整合。"（第50章）

## 分析流程

### Step 1: 材料收集

1. 确认素材来源（聊天记录/文章/决策故事/转折经历/对话记录）
2. 划分素材段落，标注每段的主题和时间线

### Step 2: 五层逐项分析

对每个层次，按照【观察】→【匹配】→【结论】的结构进行分析：

#### 【观察】— 提取原始素材中的具体表现

- 原文中说了什么？做了什么？
- 有哪些关键词/句式/行为？
- 对不同人/不同情境的反应一样吗？

#### 【匹配】— 对照理论框架和书籍案例

- 对照本层的**类型表格**，哪些类型与之匹配？
- 对照 references/cases.md 中的哪个案例？
- 用书中的哪个原理解释最合适？

#### 【结论】— 该层次的模式判断

- 此人属于哪种类型/哪个阶段？
- 置信度如何（高/中/低）？
- 需要补充什么信息才能更确定？

### Step 3: 交叉验证

1. 五个层次之间是否一致？（如：行为层的逃避可能对应思维层的防御型）
2. 哪个层次的信息最充分？哪个最薄弱？
3. 结论是否有矛盾？如有，标注待澄清

### Step 4: 输出心智画像

按照标准格式输出分析结论（见下文模板）
