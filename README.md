- **五层完整分析**：行为层 → 思维层 → 关系层 → 转折层 → 人生阶段层

## 📋 分析框架概览

### 第一层：行为层 —— 改变与行动

分析 **TA 想不想改变？是什么在阻止 TA 改变？**

| 类型 | 说明 |
|---|---|
| 心理舒适区 | 维持现状会带来安全感 |
| 改变受阻 | 想改变但迈不出步 |
| 环境场 | 特定环境下 automatism 会做某事 |
| 情感驱动 | 焦虑驱动 vs 爱驱动 |

### 第二层：思维层 —— 心智模式

分析 **TA 用什么方式看待世界和处理信息？**

| 类型 | 子类型 | 说明 |
|---|---|---|
| 成长型 | — | 能力可以成长，失败是学习机会 |
| 防御型 | 僵固型 | 特别在意"聪明"形象 |
| 防御型 | 应该思维 | 用"应该""必须"要求世界 |
| 防御型 | 绝对化 | 给判断加"永远/总是" |

### 第三层：关系层 —— 自我在关系中

分析 **TA 在关系中是什么角色？TA 如何处理关系？**

| 类型 | 说明 |
|---|---|
| 关系角色 | 照顾者/被照顾者/控制者/追随者 |
| 依恋类型 | 安全型/焦虑型/回避型 |
| 关系问题 | "都是你的错"/"都是我的错"/关系纠缠 |

### 第四层：转折层 —— 人生的转变

分析 **TA 是否经历过重大转变？现在处于哪个阶段？**

| 阶段 | 说明 |
|---|---|
| 结束期 | 旧的身份/关系/工作在结束 |
| 迷茫期 | 旧的已结束，新的还没来 |
| 重生期 | 找到了新的身份和方向 |
| 稳定期 | 暂无明显转折 |

### 第五层：人生阶段层 —— 一辈子的课题

分析 **TA 处于什么人生阶段？这个阶段的核心课题是什么？**

| 阶段 | 年龄 | 核心课题 |
|---|---|---|
| 青春期 | 12-18 | 我是谁 |
| 成年早期 | 25-35 | 亲密关系 + 职业认同 |
| 中年期 | 35-60 | 繁衍 vs 停滞 |
| 晚年 | 60+ | 整合一生 |
